# Expense Tracker with AI Chat

A modern, responsive expense tracker built with Next.js, Tailwind CSS, and Supabase, featuring an AI-powered assistant.

## features

- **Dashboard**: Visualize your spending with interactive charts (Line & Pie).
- **Add Expense**: Easily log new expenses with categories.
- **AI Chat**: A persistent chat assistant to answer questions about your finances.
- **Responsive Design**: Works on desktop and mobile.

## Setup Instructions

### 1. Prerequisites

- Node.js installed.
- A [Supabase](https://supabase.com/) account.

### 2. Database Setup (Supabase)

1.  Create a new project in Supabase.
2.  Go to the **SQL Editor** in your Supabase dashboard.
3.  Run the following SQL query to create the `expenses` table:

```sql
create table expenses (
  id bigint generated by default as identity primary key,
  created_at timestamp with time zone default timezone('utc'::text, now()) not null,
  title text not null,
  amount numeric not null,
  description text,
  category text not null,
  payment_mode text default 'Cash',
  date date not null
);

create table savings (
  id bigint generated by default as identity primary key,
  created_at timestamp with time zone default timezone('utc'::text, now()) not null,
  amount numeric not null,
  mode text not null,
  description text,
  date date not null
);
```

4.  (Optional) Disable Row Level Security (RLS) for simplicity, or add a policy to allow public insert/select if you want to test without auth.
    *   To disable RLS: Go to **Table Editor** -> `expenses` -> **RLS** -> Disable RLS.
    *   **Note**: For a real production app, you should enable RLS and implement Authentication.

### 3. Environment Variables

1.  Create a file named `.env.local` in the root of the project (`expense-tracker`).
2.  Add your Supabase credentials:

```env
NEXT_PUBLIC_SUPABASE_URL=your_supabase_project_url
NEXT_PUBLIC_SUPABASE_ANON_KEY=your_supabase_anon_key
```

You can find these in Supabase under **Project Settings** -> **API**.

### 4. Installation & Running

1.  Install dependencies:
    ```bash
    npm install
    ```

2.  Run the development server:
    ```bash
    npm run dev
    ```

3.  Open [http://localhost:3000](http://localhost:3000) in your browser.

## Project Structure

- `src/app`: Next.js App Router pages and API routes.
- `src/components`: Reusable UI components (Dashboard, Forms, Chat).
- `src/lib`: Utility functions and Supabase client.

## Technologies
